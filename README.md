# Python Beginner Projects 🐍

Beginner Python projects built during Angela Yu's
100 Days of Code bootcamp.

## Projects

### 1. Band Name Generator
Enter your city and pet name to generate a fun band name.

### 2. Tip Calculator 💰
Enter the bill amount, tip percentage, and number of people
— get the exact amount each person should pay in Rs.
- Concepts: float(), int(), round(), f-strings
